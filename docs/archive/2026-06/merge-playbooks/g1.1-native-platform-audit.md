**One concern**: `runtime_status.cpp` uses `#ifdef _WIN32` (lines 21, 48) for shell command quoting (`2>nul` vs `2>/dev/null`) and for adding `wintun_path`/`tap_installer_path` fields. The alternative would be to delegate version detection and platform-specific field injection to `src/platform/win32/` and `src/platform/darwin/` respectively. However, this is an acceptable trade-off since the `#ifdef` only affects string formatting and optional JSON fields, not core logic.
